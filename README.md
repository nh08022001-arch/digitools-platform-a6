# DigiTools

A premium digital tools platform to help students, creators, professionals, and businesses work smarter.

---

## 🚀 Technology Used

- React.js – Frontend library for building UI  
- Tailwind CSS, Daisyui – Styling and responsive design  
- React Icons – For icons  
- React Toastify – Notification alerts  
- JSON – Local product data  

---

## 🌟 Features

1. Product Catalog – Browse premium digital tools with tags and details.  
2. Cart Functionality – Add, remove, and checkout products with notifications.  
3. Responsive UI – Beautifully designed for both desktop and mobile devices.
